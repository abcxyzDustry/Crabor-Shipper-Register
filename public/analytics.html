Tên sự kiện Hướng Emit từ Nhận tại Mô tả
newShipperApplication SERVER→CLIENT server.js admin Shipper mới đăng ký → Admin nhận ngay, flash card, refresh bảng
newPartnerApplication SERVER→CLIENT server.js admin Đối tác Giặt Là / Giúp Việc / China Shop đăng ký mới
registrationStatusUpdated SERVER→CLIENT server.js admin Admin duyệt/từ chối → tất cả tab admin cùng update
newOrder SERVER→CLIENT server.js admin shippers Đơn mới tạo → Admin thấy ngay, shipper nhận thông báo
orderStatusChanged SERVER→CLIENT server.js customer_X admin Cập nhật trạng thái đơn → customer và admin cùng nhận
shipperTracking SERVER→CLIENT server.js order_X GPS shipper → customer xem bản đồ real-time
joinRoom CLIENT→SERVER admin customer server.js Client tự đăng ký vào phòng: 'admin', 'shippers', 'order_X', 'customer_X'
orderUpdate CLIENT→SERVER shipper app server.js Shipper cập nhật: đã lấy hàng, đang giao, đã giao
shipperLocation CLIENT→SERVER shipper app server.js GPS từ app shipper → server relay đến customer + admin

Hướng dẫn cài đặt và chạy
1
Cấu trúc thư mục project
CRABOR/ ├── server.js ← Server gốc (duy nhất, port 3000) ├── package.json ├── .env ← KHÔNG push GitHub ├── .env.example ├── .gitignore └── public/ ├── index.html ← Landing (chọn vai trò) ├── customer.html ← App khách hàng ├── shipper.html ← App shipper ├── partner.html ← App đối tác ├── admin.html ← Dashboard admin (real-time) ├── shipper-register.html ← Form đăng ký shipper └── partner-register.html ← Form đăng ký đối tác
2
File .env (điền đầy đủ trước khi chạy)
PORT=3000 NODE_ENV=development MONGODB_URI=mongodb+srv://123456789a:<PASS>@cluster0.fs703g5.mongodb.net/crabor SESSION_SECRET=crabor-session-secret-2025 ADMIN_SECRET_KEY=crabor-admin-secret-2025 ADMIN_DEFAULT_PASS=Crabor@2025 ESMS_API_KEY=your_esms_api_key ESMS_SECRET_KEY=your_esms_secret_key
3
Cài dependencies và chạy
# Cài packages npm install # Dev (auto-restart khi sửa code) npm run dev # Production npm start # Kiểm tra OK: ✅ MongoDB Atlas connected — DB: crabor 🚀 Server running on port 3000
4
Các URL sau khi chạy — tất cả cùng 1 server
http://localhost:3000/ ← Landing http://localhost:3000/customer ← App khách hàng http://localhost:3000/shipper ← App shipper http://localhost:3000/partner ← App đối tác http://localhost:3000/admin ← Admin real-time dashboard http://localhost:3000/shipper/register ← Form ĐK shipper (public) http://localhost:3000/partner/register ← Form ĐK đối tác (public)
5
Test kết nối Socket.io real-time
# 1. Mở Admin Dashboard: http://localhost:3000/admin → Đăng nhập → Thấy badge LIVE (xanh) # 2. Mở tab mới, vào form đăng ký: http://localhost:3000/shipper/register # 3. Điền form và submit # → Admin dashboard nhận thông báo ngay lập tức ⚡ # → Bảng shipper tự refresh không cần F5 # → Stats card flash và cập nhật số
